## 🛠️ Stack Technique
- **Frontend** : React 19, Vite, TailwindCSS, Lucide React, Framer Motion
- **Internationalisation** : i18next (Français & Anglais)
- **Backend/SaaS** : Expérience en Java (Spring Boot), Python, PostgreSQL, MySQL
- **DevOps** : Docker, Nixpacks, Coolify (Hetzner VPS), GitHub Actions

## ✨ Fonctionnalités
- 🌍 **Multilingue** : Support complet du français et de l'anglais.
- 🌓 **Mode Sombre** : Interface moderne et responsive.
- 📱 **Mobile First** : Optimisé pour tous les types d'écrans.
- 📬 **Formulaire de Contact** : Intégration directe via EmailJS.
- 📄 **Téléchargement de CV** : Accès direct aux versions FR et EN de mon CV.

## 📁 Structure du Projet
- `src/components` : Composants UI réutilisables (Hero, Projects, Experience, etc.)
- `src/i18n` : Fichiers de traduction (locales JSON)
- `public/` : Ressources statiques (PDF du CV, Favicon, Images de profil)

## 💻 Installation Locale

1. Cloner le projet :
   ```bash
   git clone https://github.com/davilla1993/portfolio.git
   ```
2. Installer les dépendances :
   ```bash
   npm install
   ```
3. Lancer le serveur de développement :
   ```bash
   npm run dev
   ```
4. Construire pour la production :
   ```bash
   npm run build
   ```

## 🏗️ Déploiement
Ce projet est configuré pour être déployé sur **Coolify** via Nixpacks.
- Le fichier `nixpacks.toml` gère les étapes de build et de démarrage.
- Les fichiers `vercel.json` et `_redirects` gèrent le routage des Single Page Applications (SPA).
